# Project Plan (PMBOK)

## Milestone 1: macOS Setup

- Verify Apple Silicon hardware (M1/M2/M3/M4)
- Install Xcode Command Line Tools
- Install Homebrew package manager
- Configure shell environment (zsh)

## Milestone 2: Metal GPU Configuration

- Verify Metal support on macOS
- Install Ollama with Metal acceleration
- Configure GPU memory allocation
- Test Metal GPU rendering with sample model

## Milestone 3: Ollama Deployment

- Pull required models (granite4, qwen-coder)
- Configure fixed ports (55077, 66044)
- Set up service startup
- Verify model inference performance
