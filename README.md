# ollama-mac

Ollama LLM deployment on macOS with Apple Silicon/Metal GPU acceleration

## Features

- Metal GPU acceleration
- macOS native
- Homebrew support

## Fixed Ports

- 55077: granite4
- 66044: qwen-coder

## Related Projects

- toolchain-module
- ollama-hpcc
- ollama-rocky
- ollama-podman

## Installation

```bash
pip install -e src/python/
```
